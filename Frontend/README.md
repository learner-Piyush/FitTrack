# FitTrack Frontend

A modern React-based frontend for the FitTrack fitness tracking application.

## Tech Stack

- React 19, Vite, React Router, Tailwind CSS
- Axios, Recharts, Lucide React, React Hot Toast, date-fns

## Installation

```bash
npm install
```

## Build

```bash
npm run build
```

## Deployment to Vercel

1. Push your code to a Git repository
2. Go to Vercel and import your repository
3. Set Root Directory to `Frontend`
4. Deploy

The `vercel.json` file handles SPA routing.

## API Base URL

Configured to connect to `https://fittrack-yf0o.onrender.com/api/v1`.
